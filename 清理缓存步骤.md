# iOS项目清理缓存步骤

## 方法1：Xcode清理（推荐）
1. 在Xcode中，点击菜单：Product → Clean Build Folder（或按 Shift+Command+K）
2. 关闭Xcode
3. 删除DerivedData文件夹：
   ```bash
   rm -rf ~/Library/Developer/Xcode/DerivedData
   ```
4. 重新打开Xcode
5. 重新编译运行项目

## 方法2：删除应用重装
1. 在模拟器或真机上，完全删除DMWork应用
2. 在Xcode中重新编译运行

## 方法3：命令行清理
```bash
cd /Users/wanglitao/Documents/octo-ios
xcodebuild clean
rm -rf ~/Library/Developer/Xcode/DerivedData
```

## 验证修改是否生效
查看实际请求的URL，应该是：
✅ https://api-test.example.com/api/v1/friend/sync
❌ https://api-test.example.com/v1/friend/sync

## 如果还是不行
检查控制台日志中的网络请求，看看实际请求的URL是什么。
